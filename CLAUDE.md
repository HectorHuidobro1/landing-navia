## Skills disponibles (Superpowers)

Este proyecto tiene Superpowers instalado. Usa estos skills antes de cada tarea:

- `/brainstorming` — antes de diseñar cualquier feature
- `/writing-plans` — antes de tocar código en tareas multi-paso
- `/test-driven-development` — antes de escribir código de implementación
- `/systematic-debugging` — ante cualquier bug o fallo de test
- `/verification-before-completion` — antes de declarar trabajo como terminado
